# plan-utvecklingstidslinje-resursallokering-vibedoc

## Syfte
Bryta ner en produkt till realistiska faser med milstolpar, resursbehov och riskkontroll.

## Input
- Produktscope
- Teamroller
- Deadline
- Risker/beroenden

## Instruktioner
Kopiera prompten nedan:

```text
Create a phased development timeline with resource allocation.

Project scope:
[insert scope]

Team:
- [roles and count]

Timeline:
[target date or duration]

Tasks:
1) Split into phases (Discovery, Build, Test, Launch, Post-launch)
2) Define milestones per phase
3) Estimate effort per milestone
4) Assign role ownership
5) List top risks + mitigations

Output format:
[Phase Timeline]
- Phase:
  - Duration:
  - Milestones:

[Resource Allocation]
- Role -> responsibility

[Risk Register]
- Risk:
  - Impact:
  - Mitigation:
```

## Output-format
- Fasplan + milstolpar
- Resursallokering
- Riskregister

## Kvalitetskriterier
- Realistisk planering
- Klar ansvarsfordelning
- Tidig riskhantering

## Varianter
- Variant A: Solo/founder mode.
- Variant B: Team mode med flera streams.

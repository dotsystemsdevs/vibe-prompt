# scaffold-kodbas-snabbstart-paket

## Syfte
Snabbt skapa en stabil grundstruktur for nya moduler, tjanster eller miniappar.

## Input
- Sprak/framework
- Funktionellt mal
- Basala krav (routes, auth, storage, etc.)

## Instruktioner
Kopiera prompten nedan:

```text
Generate a scaffold for this project task:
[insert task]

Stack:
[insert stack]

Include:
1) Minimal folder/file structure
2) Core placeholders (functions/routes/classes)
3) One sample happy-path flow
4) Notes for next 3 incremental steps

Rules:
- Keep scaffold minimal and extendable.
- Do not over-engineer.
- Add TODO markers for missing business logic.

Output format:
[Structure]
...

[Starter Code]
...

[Next 3 Steps]
1) ...
2) ...
3) ...
```

## Output-format
- Strukturforslag
- Starter code
- Nasta tre steg

## Kvalitetskriterier
- Kommer igang snabbt
- Lag startfriktion
- Tydlig riktning for vidare arbete

## Varianter
- Variant A: API scaffold.
- Variant B: UI/component scaffold.

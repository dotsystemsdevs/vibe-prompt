# projekt-vibekod-0till1-praktikpaket

## Syfte
Driva ett projekt fran ide till fungerande MVP via praktiska leveranssteg.

## Input
- Produktide
- Onskad plattform
- Malkund
- Deadline

## Instruktioner
Kopiera prompten nedan:

```text
Create a 0->1 project execution plan for this vibe coding product.

Product idea:
[insert]

Constraints:
- Platform:
- Deadline:
- Skill level:

Plan sections:
1) MVP scope (must-have only)
2) Technical slice order (frontend/backend/data)
3) Weekly milestones
4) Testing/QA plan
5) Launch checklist

Output format:
[MVP Scope]
...

[Build Order]
...

[Milestones]
Week 1...

[QA Plan]
...

[Launch Checklist]
...
```

## Output-format
- MVP-scope
- Build order
- Milstolpar + launch checklista

## Kvalitetskriterier
- Fokus pa ship, inte perfectionism
- Tydlig avgransning
- Realtidstempo mot deadline

## Varianter
- Variant A: Personal tool MVP.
- Variant B: SaaS MVP with auth + payments.

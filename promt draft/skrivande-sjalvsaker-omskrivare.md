# skrivande-sjalvsaker-omskrivare

## Syfte
Skriva om text sa den later trygg, tydlig och professionell utan defensiv eller ursaktande ton.

## Input
- Originaltext
- Onskad ton (neutral, authority, varm)
- Kanal (mail, post, pitch, memo)

## Instruktioner
Kopiera prompten nedan:

```text
Rewrite this to sound confident and clear, not apologetic:
[paste text]

Rules:
1) Preserve the original intent.
2) Remove hedging and weak phrasing unless uncertainty is required.
3) Keep it concise and direct.
4) Provide:
   - polished version
   - 3 key edits and why they improve confidence

Output format:
[Polished Version]
...

[Key Edits]
1) ...
2) ...
3) ...
```

## Output-format
- Omskriven text
- 3 nyckelandringar med motiv

## Kvalitetskriterier
- Stark och tydlig ton
- Samma budskap bevaras
- Klar att skicka/publicera

## Varianter
- Variant A: Kort social-post.
- Variant B: Formellt mail till ledning/kund.

# frontend-navigation-layout-prompt-pack

## Syfte
Bygga tydliga layout- och navigationsmonster som haller ihop hela produkten.

## Input
- Apptyp
- Informationsarkitektur
- Enhetstyper

## Instruktioner
Kopiera prompten nedan:

```text
Generate a navigation and layout pack for this app.

App:
[insert app type]

Include:
1) Responsive navbar
2) Sidebar (desktop) + drawer (mobile)
3) Breadcrumbs
4) Tabs
5) Pagination
6) Footer navigation
7) One advanced layout (split view or fixed header/body/footer)

Requirements:
- Keyboard accessible navigation
- Active-state logic
- Proper layering/z-index plan
- Route-change behaviors

Output format:
[Navigation Map]
...

[Layout Patterns]
- ...

[Interaction Rules]
- open/close:
- focus behavior:
- route transitions:
```

## Output-format
- Navigation map
- Layoutmonster
- Interaktionsregler

## Kvalitetskriterier
- Stark informationsarkitektur
- Konsistent navigation mellan vyer
- Mindre orienteringsfriktion for anvandaren

## Varianter
- Variant A: Marketing + docs site.
- Variant B: Product app with dense workflows.

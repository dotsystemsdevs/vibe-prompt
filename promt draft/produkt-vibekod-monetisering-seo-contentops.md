# produkt-vibekod-monetisering-seo-contentops

## Syfte
Koppla vibe coding-produkten till verkligt varde via monetisering, SEO och content/distribution.

## Input
- Produkttyp
- Malgrupp
- Kanalstrategi
- Intaktsmal

## Instruktioner
Kopiera prompten nedan:

```text
Design a monetization and growth plan for my AI-built product.

Inputs:
- Product:
- Target audience:
- Acquisition channels:
- Revenue target:

Tasks:
1) Propose 2-3 monetization models.
2) Define MVP pricing strategy.
3) Build SEO content plan (topic clusters + publish cadence).
4) Define distribution loop (social/community/email).
5) Add KPI dashboard (traffic, activation, conversion, retention, revenue).

Output format:
[Monetization Options]
...

[Pricing Strategy]
...

[SEO Plan]
...

[Distribution Loop]
...

[KPIs]
...
```

## Output-format
- Monetiseringsalternativ
- Prissattningsplan
- SEO + distribution
- KPI-ramverk

## Kvalitetskriterier
- Från "byggt" till "anvant och betalt"
- Datadriven tillvaxtlogik
- Hallbar go-to-market-rutin

## Varianter
- Variant A: Indie solo product.
- Variant B: B2B niche SaaS.

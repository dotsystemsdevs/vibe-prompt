# frontend-landingpage-prompt-pack

## Syfte
Snabbt generera starka landing page-sektioner med tydlig branding, struktur och CTA.

## Input
- Produkt och malgrupp
- Sektionstyp (hero, pricing, faq, testimonials, footer)
- Designconstraints

## Instruktioner
Kopiera prompten nedan:

```text
Generate a landing page section pack for this product.

Product:
[insert product]

Audience:
[insert audience]

Stack:
React + Tailwind CSS

Create these sections:
1) Hero
2) Feature grid
3) Social proof
4) Pricing with monthly/annual toggle
5) FAQ accordion
6) Final CTA

Each section must include:
- component goals
- exact content slots
- responsive behavior
- interaction states
- accessibility notes

Output format:
[Section 1: Hero]
...
[Section 2: Feature Grid]
...
...
```

## Output-format
- Sektion-for-sektion blueprint
- Responsive + interaction notes

## Kvalitetskriterier
- Konsekvent storytelling genom sidan
- Tydliga CTA-floden
- Mobilforsta designkvalitet

## Varianter
- Variant A: SaaS landing.
- Variant B: Consumer app landing.

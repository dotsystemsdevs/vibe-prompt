# workflow-session-reflection-och-breakthrough-logg

## Syfte
Bygga kumulativ teamkunskap efter varje session via reflection + breakthrough-dokumentation.

## Input
- Sessionhistorik
- Problem/fixar
- Upptackta monster

## Instruktioner
Kopiera prompten nedan:

```text
Create a post-session reflection and breakthrough documentation.

Tasks:
1) Summarize problems solved and root causes.
2) Capture patterns that should be reused.
3) Capture user/team workflow preferences.
4) Identify knowledge gaps and next improvements.
5) Document any major breakthrough with:
 - problem
 - insight
 - implementation example
 - impact metrics
 - reusable principle

Output format:
[Session Reflection]
...

[Patterns to Reuse]
...

[Breakthrough Entry]
...

[Action Items]
- AGENTS.md updates
- test additions
- docs updates
```

## Output-format
- Reflection
- Patternlogg
- Breakthrough-entry + action items

## Kvalitetskriterier
- Varje session gor nasta bättre
- Mindre upprepade misstag
- Starkare institutional memory

## Varianter
- Variant A: Solo logg.
- Variant B: Team knowledge base logg.

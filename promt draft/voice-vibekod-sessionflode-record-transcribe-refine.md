# voice-vibekod-sessionflode-record-transcribe-refine

## Syfte
Standardisera voice coding-sessioner från inspelning till färdig prompt med minimal friktion.

## Input
- Sessionmål
- Inspelad transkription
- Projektkontext

## Instruktioner
Kopiera prompten nedan:

```text
Run this voice coding session workflow:
Record -> Transcribe -> Refine -> Execute

Session goal:
[insert goal]

Transcript:
[insert transcript]

Project context:
[insert context]

Tasks:
1) Produce cleaned transcript.
2) Produce structured coding prompt.
3) Produce execution checklist.
4) Produce post-run reflection notes.

Output format:
[Clean Transcript]
...

[Structured Prompt]
...

[Execution Checklist]
- ...

[Post-Run Notes]
- ...
```

## Output-format
- Clean transcript
- Strukturerad prompt
- Checklista + post-run noteringar

## Kvalitetskriterier
- Konsekvent sessionkvalitet
- Mindre kontexttapp
- Snabbare från idé till handling

## Varianter
- Variant A: Solo coding sprint.
- Variant B: Pair/mentor voice session.

# workflow-promptbibliotek-11tillobegransat-styrning

## Syfte
Skapa en prompt for att strukturera promptbibliotek fran liten basuppsattning till obegransad samling utan kaos.

## Input
- Nuvarande promptlista
- Antal aktiva arbetsomraden
- Om laget ar free eller pro

## Instruktioner
1. Kategorisera prompts efter jobbtyp (build, debug, review, docs, deploy).
2. Satt naming-konvention och metadata for snabb sokbarhet.
3. Definiera vilka prompts som alltid ska vara "top pinned".
4. Foresla nar en prompt ska slas ihop, delas eller arkiveras.
5. Om pro-lage finns: skapa skalbar strategi for obegransat bibliotek.
6. Leverera en underhallsrutin varje vecka for bibliotekskvalitet.

## Output-format
- `Library Taxonomy`
- `Prompt Naming Rules`
- `Pinned Set`
- `Scale Strategy`
- `Weekly Maintenance Routine`

## Kvalitetskriterier
- Enkel att hitta ratt prompt snabbt
- Forhindrar duplicat och promptskrot
- Fungerar i bade free- och pro-scenario
- Latt att overlamna till andra

## Varianter
- Variant A: Minimal samling for nyborjare
- Variant B: Stort professionellt bibliotek med versionsdisciplin

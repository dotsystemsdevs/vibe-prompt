# karriar-promptengineer-till-llmengineer-roadmap

## Syfte
Skapa en realistisk larandefoljd for overgangen fran prompt engineer till AI/LLM engineer med tydliga etapper.

## Input
- Nuvarande erfarenhet
- Veckotid
- Malroll (AI Engineer, LLM Engineer, AI Platform Engineer)
- Tidshorisont (3-12 manader)

## Instruktioner
Kopiera prompten nedan:

```text
You are a technical career coach for AI engineering transitions.
Build a realistic learning path from Prompt Engineer to AI/LLM Engineer.

Inputs:
- Current experience:
- Weekly hours:
- Target role:
- Timeline:

Required skill domains:
- Python (services + async)
- SQL + databases
- Git workflows
- Docker
- RAG engineering end-to-end
- LLM evaluation and regression testing
- Production monitoring and observability
- Security and PII handling

Tasks:
1) Define learning order with prerequisites.
2) Split path into phases with duration:
   - Foundation
   - Build
   - Productionization
   - Interview readiness
3) Add code-first milestones for each phase.
4) Include "stop doing" advice (common time-wasters).

Output format:
<TransitionPlan>
Phase 1 ...
Phase 2 ...
Phase 3 ...
Phase 4 ...
</TransitionPlan>

<Milestones>
- Milestone:
  - Deliverable:
  - Proof of skill:
</Milestones>

<RiskControls>
- Likely blockers
- Mitigations
</RiskControls>
```

## Output-format
- Fasindelad transition-plan
- Kodbara milstolpar
- Riskkontroller for att halla kurs

## Kvalitetskriterier
- Realistisk i tid och scope
- Praktisk kodproduktion varje fas
- Tydlig ordning mellan beroenden
- Fokus pa employability

## Varianter
- Variant A: Snabbspår till junior AI engineer (3-4 manader).
- Variant B: Stabil senior-track (9-12 manader med djupare systemsyn).

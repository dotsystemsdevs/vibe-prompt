# larande-roadmap-rollbaserad-utvecklare

## Syfte
Ge en komplett laranderoadmap for frontend, backend eller full-stack med tydlig ordning och prioritering.

## Input
- Onskad roll (Frontend/Backend/Full-Stack)
- Startniva
- Tillganglig tid per vecka
- Onskad tidshorisont (3, 6, 12 manader)

## Instruktioner
Kopiera prompten nedan:

```text
You are a software career roadmap coach.
Create a complete roadmap to become a [Frontend/Backend/Full-Stack] developer.

Inputs:
- Current level:
- Weekly hours:
- Timeline:

Requirements:
1) Split into Beginner, Intermediate, Advanced.
2) For each stage provide:
   - Core topics
   - Recommended build projects
   - Common mistakes to avoid
   - Exit criteria (how to know you are ready for next stage)
3) Show learning order with dependencies (what must come first).
4) Include portfolio priorities and interview prep checkpoints.
5) Keep it practical, not academic-heavy.

Output format:
<RoadmapOverview>
[Stage summary + time estimate]
</RoadmapOverview>

<StageDetails>
Beginner: ...
Intermediate: ...
Advanced: ...
</StageDetails>

<PortfolioPlan>
- Project 1
- Project 2
- Project 3
</PortfolioPlan>
```

## Output-format
- Trestegs-roadmap
- Klar ordning mellan skills
- Portfolioplan med prioritet

## Kvalitetskriterier
- Tydlig progression
- Praktiska projekt i varje steg
- Minskar "vad ska jag gora nu?"-stress
- Matchar faktisk rollkrav-bild

## Varianter
- Variant A: Job-ready pa 6 manader (fokus pa portfolio + intervjuer).
- Variant B: Deep mastery pa 12 manader (fler internals och systemkunskap).

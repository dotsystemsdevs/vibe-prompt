# docs-aifirst-prd-generator-smartscope

## Syfte
Skapa en komplett, testbar PRD som minskar AI-hallucinationer och scope creep.

## Input
- App-idé
- Målgrupp
- Problem
- Affärsmål

## Instruktioner
Kopiera prompten nedan:

```text
Create an AI-first PRD for this app idea.

Context:
- Idea:
- Target users:
- Core problem:
- Business objective:

Must include:
1) Product overview and problem statement
2) SMART goals and success metrics
3) Personas (primary + secondary)
4) Feature tiers (P0/P1/P2)
5) Explicit out-of-scope list
6) User stories + acceptance criteria
7) Non-functional requirements
8) Risks, assumptions, dependencies
9) MVP definition of done

Rules:
- Requirements must be specific and testable.
- Focus on WHAT, not implementation HOW.
- Include edge cases and error states.

Output format:
[PRD]
...
```

## Output-format
- Full PRD i markdown

## Kvalitetskriterier
- Mätbar och testbar
- Scope tydligt avgränsat
- Klar för implementationplanering

## Varianter
- Variant A: Solo MVP.
- Variant B: Team product launch.

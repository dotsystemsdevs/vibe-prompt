# workflow-ai-verktygsval-efter-profil-och-mal

## Syfte
Valja ratt AI-verktygskombo beroende pa erfarenhet, budget, projekt och hastighet.

## Input
- Anvandarprofil
- Projektkategori
- Budget/tidskrav
- Kodvana

## Instruktioner
Kopiera prompten nedan:

```text
Recommend an AI coding tool stack for my profile and goals.

Inputs:
- Experience:
- Project type:
- Budget:
- Speed vs control preference:

Evaluate categories:
1) Model choice
2) AI editor/IDE
3) Agent platform
4) CLI tools
5) Optional no-code builder

Return:
- Best stack (primary + backup)
- Why this stack fits me
- 7-day onboarding plan
- Cost-saving tips

Output format:
[Recommended Stack]
...

[Why This Fits]
...

[7-Day Onboarding]
Day 1...

[Cost Optimization]
...
```

## Output-format
- Primar + backup stack
- 7-dagars onboarding
- Kostnadsoptimering

## Kvalitetskriterier
- Matchar verklig profil
- Undviker verktygs-overload
- Snabb time-to-output

## Varianter
- Variant A: Solo beginner.
- Variant B: Team/pro workflow.

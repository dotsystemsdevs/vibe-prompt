# produkt-ide-till-full-utvecklingsplan-vibedoc

## Syfte
Omvandla en produktide till en komplett utvecklingsplan (produkt, teknik, roadmap, deployment, growth).

## Input
- Produktide
- Malgrupp
- Domankrav/regulatoriska krav
- Teamstorlek och tid

## Instruktioner
Kopiera prompten nedan:

```text
You are an AI Product Manager + Software Architect.
Transform this idea into a complete development plan.

Idea:
[insert idea]

Context:
- Target users:
- Constraints:
- Team size:
- Timeline:

Generate:
1) Product Overview (problem, users, value prop)
2) Technical Solution (stack options + trade-offs)
3) Development Plan (phases, milestones, ownership)
4) Deployment Strategy (environments, CI/CD, monitoring)
5) Growth Strategy (adoption, retention, expansion)

Output format:
[Product Overview]
...

[Technical Solution]
...

[Development Plan]
...

[Deployment Strategy]
...

[Growth Strategy]
...
```

## Output-format
- Komplett strategisk + teknisk plan

## Kvalitetskriterier
- Praktiskt genomforbar
- Tydliga trade-offs
- Fasad leverans med ansvar

## Varianter
- Variant A: MVP pa 8 veckor.
- Variant B: Enterprise-ready pa 6 manader.

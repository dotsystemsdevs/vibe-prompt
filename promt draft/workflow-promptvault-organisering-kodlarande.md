# workflow-promptvault-organisering-kodlarande

## Syfte
Organisera kodlarande-prompts i ett "prompt vault" sa de gar att hitta, ateranvanda och iterera.

## Input
- Nuvarande promptlista
- Kategorier som anvands
- Vanliga use cases

## Instruktioner
Kopiera prompten nedan:

```text
Help me build a clean prompt vault for coding-learning prompts.

Tasks:
1) Propose folder categories:
   - planning
   - roadmap
   - projects
   - debugging
   - concept learning
2) Create a file naming standard.
3) Define metadata tags for each prompt:
   - level
   - language
   - outcome type
   - tested date
4) Suggest deduplication rules to avoid prompt clutter.
5) Add a weekly review checklist.

Output format:
[Vault Structure]
...

[Naming Standard]
...

[Metadata Tags]
...

[Dedup Rules]
...

[Weekly Checklist]
...
```

## Output-format
- Vaultstruktur
- Namn- och taggstandard
- Dedup + veckoreview

## Kvalitetskriterier
- Snabb sokbarhet
- Mindre duplicat
- Enkel forvaltning over tid

## Varianter
- Variant A: Solo vault.
- Variant B: Team vault med reviewansvar.

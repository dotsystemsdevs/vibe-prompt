# workflow-frontend-promptbibliotek-folstruktur

## Syfte
Bygga ett ateranvandbart frontend promptbibliotek sa du kan grab-tweak-ship pa minuter.

## Input
- Nuvarande prompts
- Tech stack
- Vanliga komponenttyper

## Instruktioner
Kopiera prompten nedan:

```text
Design a frontend prompt library structure for fast reuse.

Create folders:
- landing/
- dashboard/
- forms/
- navigation/
- debug/
- utilities/

For each prompt file include metadata:
- purpose
- stack
- required inputs
- output format
- test status
- known failure modes

Add governance:
1) naming convention
2) versioning rule
3) monthly cleanup checklist
4) duplicate detection rule

Output format:
[Folder Structure]
...

[Prompt Metadata Template]
...

[Maintenance Rules]
...
```

## Output-format
- Folderstruktur
- Metadatamall
- Underhallsregler

## Kvalitetskriterier
- Hittar ratt prompt snabbt
- Mindre duplicat och brus
- Stabil promptkvalitet over tid

## Varianter
- Variant A: Solo creator.
- Variant B: Team with PR review flow.

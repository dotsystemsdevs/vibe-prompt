# workflow-claudecode-kommandocheatsheet-generator

## Syfte
Skapa en personlig command cheat sheet for snabbare och mer konsekvent arbete i Claude Code-liknande floden.

## Input
- Vanliga uppgiftstyper
- Teamets verktyg/testkommandon
- Vanliga misslyckanden

## Instruktioner
Kopiera prompten nedan:

```text
Generate a practical command cheat sheet for my AI coding workflow.

Categories to include:
1) Explore codebase
2) Implement feature
3) Debug errors
4) Refactor safely
5) Test and verify
6) Pre-commit quality checks
7) Incident recovery commands

For each command pattern provide:
- When to use
- Exact command/prompt style
- Common failure mode
- Quick recovery step

Output format:
[Cheat Sheet]
## Category
- Command pattern:
- Use when:
- Watch out:
- Recovery:
```

## Output-format
- Kategoriserad cheatsheet
- Failure + recovery per kommando

## Kvalitetskriterier
- Snabb att scanna
- Byggd pa verkliga arbetsmoment
- Minskar trial-and-error

## Varianter
- Variant A: Solo setup.
- Variant B: Team setup med standardiserade gates.

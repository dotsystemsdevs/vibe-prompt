# workflow-promptbibliotek-100plus-system

## Syfte
Hantera stora promptbibliotek (100+ prompts) utan att det blir Notion/notes-kaos.

## Input
- Befintliga prompts
- Kategorier
- Testhistorik

## Instruktioner
Kopiera prompten nedan:

```text
Design an operating system for a 100+ prompt library.

Must include:
1) Folder/category taxonomy
2) Naming convention
3) Metadata schema
4) Test status lifecycle
5) Archiving policy
6) Retrieval rules (how to find best prompt fast)

Metadata fields:
- purpose
- input shape
- output format
- model fit
- tested date
- success rate
- version

Output format:
[Library Architecture]
...

[Lifecycle]
- Draft -> Tested -> Stable -> Deprecated -> Archived

[Retrieval Playbook]
- If task is X, search Y first
```

## Output-format
- Biblioteksarkitektur
- Prompt-livscykel
- Sok- och valplaybook

## Kvalitetskriterier
- Hog ateranvandning
- Mindre duplicat
- Enkel forvaltning over tid

## Varianter
- Variant A: Markdown+Git.
- Variant B: Notion/DB med metadatafilters.

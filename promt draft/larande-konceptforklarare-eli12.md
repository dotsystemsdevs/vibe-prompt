# larande-konceptforklarare-eli12

## Syfte
Forenkla komplexa kodkoncept med analogier och exempel utan att tappa korrekthet.

## Input
- Koncept (t.ex. recursion, closures, async/await)
- Sprak (valfritt)
- Nuvarande niva

## Instruktioner
Kopiera prompten nedan:

```text
Explain [Coding Concept] like I am 12 years old.
Use simple language, a relatable analogy, and one practical code example.

Then add:
1) "What this is NOT" to prevent confusion.
2) 3 quick quiz questions.
3) A tiny practice task I can do in 10 minutes.

Constraints:
- No jargon unless you define it in one sentence.
- Keep explanation concise but accurate.
- If using code, keep it short and readable.

Output format:
<SimpleExplanation>
...
</SimpleExplanation>

<Analogy>
...
</Analogy>

<MiniExample>
...
</MiniExample>

<CommonConfusions>
- ...
</CommonConfusions>

<QuickQuiz>
1) ...
2) ...
3) ...
</QuickQuiz>

<10MinuteTask>
...
</10MinuteTask>
```

## Output-format
- Enkel forklaring
- Analogi + mini-exempel
- Quiz + miniuppgift

## Kvalitetskriterier
- Latt att forsta pa forsta lasningen
- Fortfarande tekniskt korrekt
- Inbyggd sjalvtestning
- Driver faktisk inlarning

## Varianter
- Variant A: Superkort (under 150 ord + 1 kodsnutt).
- Variant B: Djupare med "nar detta bryter ihop" och vanliga fallgropar.

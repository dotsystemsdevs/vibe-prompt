# arkitektur-mermaid-diagramgenerator-vibedoc

## Syfte
Skapa professionella Mermaid-diagram for arkitektur, business flow och tidslinje direkt fran textkrav.

## Input
- Systemkomponenter
- Datafloden
- Processer/milstolpar

## Instruktioner
Kopiera prompten nedan:

```text
Generate Mermaid diagrams from this system description.

System description:
[insert description]

Create:
1) System Architecture diagram
2) Business Flowchart
3) Gantt timeline (phases + milestones)

Rules:
- Keep labels concise
- Use clear directional flows
- Include external integrations and data stores

Output format:
[Architecture Mermaid]
```mermaid
...
```

[Business Flow Mermaid]
```mermaid
...
```

[Gantt Mermaid]
```mermaid
...
```
```

## Output-format
- 3 Mermaid-diagram redo att klistra in docs

## Kvalitetskriterier
- Lattlasta diagram
- Matchar textkrav och scope
- Anvandbara i tech/docs/pitch

## Varianter
- Variant A: Monolith/system.
- Variant B: Microservices + externa APIs.

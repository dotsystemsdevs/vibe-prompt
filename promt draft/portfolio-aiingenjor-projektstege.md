# portfolio-aiingenjor-projektstege

## Syfte
Generera en projektstege som bygger verkliga AI/LLM-engineering-fardigheter och samtidigt fungerar som portfolio.

## Input
- Nuvarande niva
- Tillganglig tid
- Malroll
- Foredragen doman (fintech, retail, support, health, etc.)

## Instruktioner
Kopiera prompten nedan:

```text
You are a portfolio mentor for aspiring AI/LLM engineers.
Create a sequence of 4-6 portfolio projects that progressively build job-ready skills.

Must cover:
- Python service development (FastAPI, async basics)
- SQL + data modeling
- Dockerized deployment
- End-to-end RAG
- Evaluation and regression testing
- Monitoring and observability

For each project include:
1) Objective
2) Required stack
3) Core implementation tasks
4) "Hiring signal" (what recruiters/interviewers will infer)
5) Demo checklist and README outline

Output format:
<ProjectSequence>
Project 1 ...
Project 2 ...
...
</ProjectSequence>

<SkillCoverageMap>
- Skill -> Which project proves it
</SkillCoverageMap>

<InterviewNarrative>
- Storyline for presenting this portfolio in interviews
</InterviewNarrative>
```

## Output-format
- Progressiv projektsekvens
- Skill coverage map
- Intervjunarrativ for presentation

## Kvalitetskriterier
- Varje projekt bevisar tydlig kompetens
- Bra balans mellan bredd och djup
- Kan visas upp med demo + kod + dokumentation
- Matchar faktiska jobbkrav

## Varianter
- Variant A: 3-manaders intensiv portfolio.
- Variant B: 6-manaders djupportfolio med fler prod-aspekter.

# prompting-kontrastutkast-safe-vs-risk

## Syfte
Generera tva medvetet kontrasterade utkast for att snabbare hitta en stark mittpunkt.

## Input
- Uppgift (copy, strategi, UX-text, tekniskt forslag)
- Malgrupp
- Ton
- Risknivagrans

## Instruktioner
Kopiera prompten nedan:

```text
Generate two contrasting drafts for the same objective.

Objective:
[insert objective]

Draft A:
- safe, conventional, low risk

Draft B:
- high-contrast, creative risk, still useful

Then provide:
1) strengths of each draft
2) weaknesses of each draft
3) a merged "best of both" final draft

Output format:
[Draft A]
...

[Draft B]
...

[Comparison]
- A strengths:
- A weaknesses:
- B strengths:
- B weaknesses:

[Merged Final]
...
```

## Output-format
- Draft A och B
- Jämforelse
- Merged final

## Kvalitetskriterier
- Tydlig kontrast
- Enkel att evaluera
- Hoger kvalitet i slutversion

## Varianter
- Variant A: Headlines/pitch.
- Variant B: Arkitekturforslag med konservativ vs aggressiv design.

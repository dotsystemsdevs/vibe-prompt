# frontend-form-input-prompt-pack

## Syfte
Skapa robusta formular/input-komponenter med bra UX, validering och tillganglighet.

## Input
- Formtyp
- Falt och valideringsregler
- Onskad interaktion

## Instruktioner
Kopiera prompten nedan:

```text
Generate a form/input component pack.

Form context:
[insert context]

Stack:
React + Tailwind + React Hook Form + Zod

Include:
1) Multi-step form pattern
2) Real-time validation behavior
3) Success/error/loading states
4) Keyboard accessibility
5) Reusable input primitives (text, select, date, upload, tags)

Requirements:
- Validate on blur and submit
- Clear field-level messages
- Prevent layout shift on errors
- Mobile-friendly controls

Output format:
[Form Architecture]
...

[Validation Rules]
...

[UI States]
- default:
- error:
- success:
- loading:

[Reusable Primitives]
- ...
```

## Output-format
- Formarkitektur
- Valideringsspec
- UI-state rules + inputprimitives

## Kvalitetskriterier
- Mindre trasig form-UX
- Hoger completion rate
- Enkelt att ateranvanda i flera floden

## Varianter
- Variant A: Signup/onboarding.
- Variant B: Checkout/payment.

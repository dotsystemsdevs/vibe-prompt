# tooling-cloudbuilders-idea-till-deploy-jamforelse

## Syfte
Skapa en prompt som hjalper dig valja ratt cloud-baserad builder for att ga fran ide till deploy med minsta mojliga friktion.

## Input
- Produktide
- Tekniska krav (frontend/fullstack/mobile)
- Tid till demo och budget

## Instruktioner
1. Klassificera behovet: snabb prototyp, intern app eller kundklar produkt.
2. Jamfor cloud builders pa kriterier: hastighet, kontroll, exportbarhet, deployflode.
3. Rekommendera ett primary-val och ett backup-val.
4. Beskriv en stegvis plan for forsta leverans inom 24-72 timmar.
5. Lista risker med lock-in och hur de minimeras.
6. Avsluta med en copy-ready startprompt for valt verktyg.

## Output-format
- `Project Profile`
- `Tool Comparison Matrix`
- `Recommended Path`
- `72h Delivery Plan`
- `Lock-in Mitigation`
- `Starter Prompt`

## Kvalitetskriterier
- Beslutet ar forankrat i verkliga constraints
- Fokus pa shipping, inte bara featurelista
- Tydlig fallback vid verktygsproblem
- Direkt anvandbar i praktiken

## Varianter
- Variant A: MVP-forst med maximal fart
- Variant B: Foundation-forst med battre langsiktig kontroll

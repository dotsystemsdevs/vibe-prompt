# larande-7dagars-kickstart-kodning

## Syfte
Skapa snabb start med tydlig momentum for personer som fastnar innan de kommer igang.

## Input
- Sprak/framework
- Tid per dag
- Slutmal efter 7 dagar

## Instruktioner
Kopiera prompten nedan:

```text
Create a 7-day coding kickstart for [language/framework].
I want momentum fast, with daily building.

Constraints:
- Time per day: [X minutes]
- Skill level: beginner
- Goal by day 7: [goal]

Requirements:
1) One focused concept per day.
2) One tiny coding task per day.
3) One "confidence checkpoint" each day.
4) Day 7 must produce a small demo/project.

Output format:
[Day 1]
- Concept:
- Task:
- Checkpoint:
...
[Day 7]
- Demo scope:
- What success looks like:
```

## Output-format
- Daglig 7-dagarsplan
- Smauppgifter + confidence checkpoints
- Demo pa dag 7

## Kvalitetskriterier
- Snabb start utan overload
- Daglig byggvana
- Tydligt mal inom en vecka

## Varianter
- Variant A: 20 min/dag.
- Variant B: 60 min/dag med storre slutdemo.

# frontend-dashboard-admin-prompt-pack

## Syfte
Bygga dashboards/adminpaneler med datamönster, tabeller, navigation och real-world states.

## Input
- Domankontext
- Datatyper/KPI:er
- Roller och permissions

## Instruktioner
Kopiera prompten nedan:

```text
Generate an admin dashboard UI pack.

Context:
[insert context]

Stack:
React + Tailwind + Recharts + TanStack Table

Include:
1) KPI overview cards
2) Primary chart area
3) Data table with actions
4) Sidebar navigation with active states
5) Activity feed
6) Empty/loading/error states

Requirements:
- Mobile and desktop behavior
- Accessible table/actions
- Clear visual hierarchy
- Mock data shape examples

Output format:
[Dashboard Blueprint]
...

[Component Specs]
- KPI cards:
- Charts:
- Table:
- Sidebar:
- Feed:

[State Handling]
- loading:
- empty:
- error:
```

## Output-format
- Dashboard blueprint
- Componentspecs
- State handling-spec

## Kvalitetskriterier
- Datatunga vyer utan kaos
- Skalbar komponentstruktur
- Bra defaults for verkliga driftfall

## Varianter
- Variant A: E-commerce admin.
- Variant B: SaaS analytics.

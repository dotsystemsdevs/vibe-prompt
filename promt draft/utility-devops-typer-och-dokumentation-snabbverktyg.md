# utility-devops-typer-och-dokumentation-snabbverktyg

## Syfte
Fa praktiska utility-leverabler snabbt (Dockerfile, typer, pipeline-outline, scripts, docs).

## Input
- Projektkontext
- Onskad utility-artefakt
- Krav/constraints

## Instruktioner
Kopiera prompten nedan:

```text
Create this practical utility artifact:
[insert artifact request]

Possible artifact types:
- Dockerfile
- Type definitions
- ML pipeline outline
- Bash->Python conversion
- Pseudocode docs

Requirements:
1) Provide sensible defaults.
2) Explain critical config choices briefly.
3) Add quick validation steps.

Output format:
[Artifact]
...

[Key Choices]
- ...

[Validation Steps]
1) ...
2) ...
```

## Output-format
- Fardig utility-artefakt
- Kort motivering av val
- Verifieringssteg

## Kvalitetskriterier
- Hog praktisk nytta
- Kort time-to-value
- Latt att validera

## Varianter
- Variant A: Devops utility.
- Variant B: Data/app utility.

# larande-vibekod-30artiklar-studiecirkelplan

## Syfte
Skapa en prompt som omvandlar en lang lista av vibe-coding-artiklar till en strukturerad studiecirkel med praktisk output.

## Input
- Artikellista
- Deltagarnas niva (nyborjare/intermediate/avancerad)
- Tillganglig tid per vecka

## Instruktioner
1. Dela upp artiklar i teman: filosofi, praktisk prompting, risker, case studies.
2. Prioritera lasordning efter larvarde och relevans.
3. Skapa veckoschema med lasa -> diskutera -> bygga mini-ovning.
4. Definiera reflektionsfragor for varje tema.
5. Krav pa konkret output varje vecka (prompt, kod, eller review-dokument).
6. Avsluta med slutprojekt och evalueringskriterier.

## Output-format
- `Theme Map`
- `Reading Priority`
- `Weekly Study Plan`
- `Discussion Questions`
- `Weekly Deliverables`
- `Final Project Rubric`

## Kvalitetskriterier
- Larandet ar hands-on och inte bara lasning
- Tydlig progression vecka for vecka
- Matbara delresultat
- Fungerar for grupper med blandad erfarenhet

## Varianter
- Variant A: 4-veckors intensiv plan
- Variant B: 8-veckors djup plan med fler praktiska sprintar

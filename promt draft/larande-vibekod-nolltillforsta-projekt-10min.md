# larande-vibekod-nolltillforsta-projekt-10min

## Syfte
Ta en total nyborjare fran noll till forsta fungerande AI-byggda miniapp pa 10-60 minuter.

## Input
- Niva (0 erfarenhet / lite erfarenhet)
- Onskat mini-projekt
- Tillgangliga verktyg

## Instruktioner
Kopiera prompten nedan:

```text
You are a beginner-first vibe coding coach.
My goal is to ship a first working mini-project quickly.

Profile:
- Experience level:
- Time available:
- Preferred tool:
- Mini-project idea:

Tasks:
1) Explain the absolute minimum setup.
2) Give a step-by-step plan (10-60 min total).
3) Provide copy-paste prompts for each step.
4) Add a "stuck recovery" section for common failures.
5) End with what to learn next in week 1.

Output format:
[Quick Setup]
...

[Step Plan]
Step 1 ...
Step 2 ...

[Copy-Paste Prompts]
...

[If Stuck]
...

[Week 1 Next Steps]
...
```

## Output-format
- Snabbstart
- Stegplan
- Promptpaket
- Recovery + nasta steg

## Kvalitetskriterier
- Noll friktion
- Snabb "first win"
- Tydlig fortsattningsvag

## Varianter
- Variant A: No-code/low-code path.
- Variant B: Code-editor path.

# research-multiperspektiv-sidobyside

## Syfte
Forbattra resonemangskvalitet genom att jamfora flera perspektiv parallellt pa samma fraga.

## Input
- Fraga/amne
- Perspektiv (t.ex. roll, ideologi, funktion)
- Beslutskriterier

## Instruktioner
Kopiera prompten nedan:

```text
Answer the same question from multiple perspectives side by side.

Question:
[insert]

Perspectives:
- Perspective A:
- Perspective B:
- Perspective C:

Requirements:
1) Each perspective must include strongest argument, biggest concern, and recommended action.
2) Then synthesize common ground and key disagreements.
3) End with a neutral recommendation given my goal.

Output format:
[Perspective A]
- Strongest argument:
- Biggest concern:
- Recommended action:

[Perspective B]
...

[Perspective C]
...

[Synthesis]
- Common ground:
- Key disagreements:

[Neutral Recommendation]
...
```

## Output-format
- Perspektivjmf side-by-side
- Syntes av samforstand och konflikt
- Neutral rekommendation

## Kvalitetskriterier
- Mindre enogd analys
- Starkare beslut under osakerhet
- Tydlig jamforbar struktur

## Varianter
- Variant A: Policy/politik.
- Variant B: Produkt- eller teknikval.

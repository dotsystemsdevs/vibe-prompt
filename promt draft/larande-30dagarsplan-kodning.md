# larande-30dagarsplan-kodning

## Syfte
Skapa en realistisk 30-dagars studieplan for ett programmeringssprak med dagliga uppgifter, resurser och mini-projekt.

## Input
- Programmeringssprak eller framework
- Nuvarande niva (nyborjare/intermediate)
- Tid per dag (t.ex. 45 min)
- Mal (jobb, hobby, bygga viss app)

## Instruktioner
Kopiera prompten nedan:

```text
You are a coding learning planner.
Create a realistic 30-day plan to learn [Programming Language/Framework].

Student profile:
- Current level: [level]
- Time per day: [minutes]
- Goal: [goal]

Requirements:
1) Break the 30 days into weekly themes.
2) For each day include:
   - Focus concept
   - 1 learning resource type (docs/video/article)
   - 1 short practical exercise
3) Include 4 mini-project milestones (end of each week).
4) Add one review day per week for spaced repetition.
5) Keep difficulty progressive and realistic.

Output format:
<30DayPlan>
Day 1: ...
...
Day 30: ...
</30DayPlan>

<WeeklyMilestones>
Week 1: ...
Week 2: ...
Week 3: ...
Week 4: ...
</WeeklyMilestones>

<SuccessCriteria>
- What "on track" looks like each week
</SuccessCriteria>
```

## Output-format
- Dag-for-dag-plan i 30 dagar
- Veckomilstolpar
- Tydliga framgangskriterier

## Kvalitetskriterier
- Realistisk arbetsmangd
- Progressiv svarighetsgrad
- Hands-on varje dag
- Inbyggd repetition

## Varianter
- Variant A: Intensiv (2-3h/dag) med fler projektleverabler.
- Variant B: Latt (20-30 min/dag) med fokus pa kontinuitet.

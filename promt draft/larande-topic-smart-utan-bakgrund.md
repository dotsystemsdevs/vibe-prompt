# larande-topic-smart-utan-bakgrund

## Syfte
Lara sig ett nytt amne snabbt med tydlig analogi och minsta nodvandiga kunskap for att prata vettigt om det.

## Input
- Topic
- Onskad niva (superbasic/basic/intermediate intro)

## Instruktioner
Kopiera prompten nedan:

```text
Explain [topic] like I am smart but have zero background.

Requirements:
1) Use one real-world analogy.
2) Give the 3 things I must know to not sound clueless.
3) Include one common misunderstanding and correction.
4) End with a 60-second summary.

Output format:
[Simple Explanation]
...

[Analogy]
...

[3 Must-Know Points]
1) ...
2) ...
3) ...

[Common Mistake]
...

[60-Second Summary]
...
```

## Output-format
- Enkel forklaring
- Analogi
- 3 must-know points
- Vanlig missuppfattning + kort summering

## Kvalitetskriterier
- Hog begriplighet
- Latt att aterberatta
- Ingen onodig jargong

## Varianter
- Variant A: ELI12.
- Variant B: "Brief for professionals" pa 150-200 ord.

# Frontend Dashboard Admin Prompt Pack

## Purpose
Build dashboards/admin panels with data patterns, tables, navigation and real-world states.

## Input
- Sentence context
- Data types/KPIs
- Roles and permissions

## Instructions
Copy the prompt below:

```text
Generate an admin dashboard UI pack.

Context:
[insert context]

Stack:
React + Tailwind + Recharts + TanStack Table

Include:
1) KPI overview cards
2) Primary chart area
3) Data table with actions
4) Sidebar navigation with active states
5) Activity feed
6) Empty/loading/error states

Requirements:
- Mobile and desktop behavior
- Accessible table/actions
- Clear visual hierarchy
- Mock data shape examples

Output format:
[Dashboard Blueprint]
...

[Component Specs]
- KPI cards:
- Charts:
- Table:
- Sidebar:
- Feed:

[State Handling]
- loading:
- empty:
- error:
```

## Output Format
- Dashboard blueprint
- Componentspecs
- State handling-spec

## Quality Criteria
- Data-heavy views without chaos
- Scalable component structure
- Good defaults for real operating cases

## Variants
- Variant A: E-commerce admin.
- Variant B: SaaS analytics.

# Frontend Ui Engineer Production Standard

## Purpose
Generate production-ready front-end solutions with a focus on readability, typing, a11y, performance and backend-agnostic integration.

## Input
- UI requirements
- Framework/stack
- API contract

## Instructions
Copy the prompt below:

```text
Act as a production UI engineer.

Requirements:
[insert requirements]

Standards:
1) Clear component architecture
2) Type-safe interfaces
3) Responsive + accessible design
4) Loading/error/empty states
5) Performance-conscious rendering
6) API-agnostic data abstraction

Output format:
[Architecture]
...

[Component Plan]
...

[Type Interfaces]
...

[Implementation]
...

[Quality Checklist]
- a11y
- performance
- testability
```

## Output Format
- Arkitektur + komponentplan
- Typinterfaces
- Implementation + quality checklist

## Quality Criteria
- Production-standard frontend
- Easy to entertain
- Smooth backend integration

## Variants
- Variant A: React + Tailwind.
- Variant B: Vue + TypeScript.

# Frontend Landingpage Prompt Pack

## Purpose
Quickly generate strong landing page sections with clear branding, structure and CTA.

## Input
- Product and template group
- Section type (hero, pricing, faq, testimonials, footer)
- Design constraints

## Instructions
Copy the prompt below:

```text
Generate a landing page section pack for this product.

Product:
[insert product]

Audience:
[insert audience]

Stack:
React + Tailwind CSS

Create these sections:
1) Hero
2) Feature grid
3) Social proof
4) Pricing with monthly/annual toggle
5) FAQ accordion
6) Final CTA

Each section must include:
- component goals
- exact content slots
- responsive behavior
- interaction states
- accessibility notes

Output format:
[Section 1: Hero]
...
[Section 2: Feature Grid]
...
...
```

## Output Format
- Sektion-for-sektion blueprint
- Responsive + interaction notes

## Quality Criteria
- Consistent storytelling throughout the page
- Clear CTA flow
- Mobile first design quality

## Variants
- Variant A: SaaS landing.
- Variant B: Consumer app landing.

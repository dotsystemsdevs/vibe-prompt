# Codebase Scaffold Quickstart Package

## Purpose
Quickly create a stable basic structure for new modules, services or mini-apps.

## Input
- Language/framework
- Functional moth
- Basic requirements (routes, auth, storage, etc.)

## Instructions
Copy the prompt below:

```text
Generate a scaffold for this project task:
[insert task]

Stack:
[insert stack]

Include:
1) Minimal folder/file structure
2) Core placeholders (functions/routes/classes)
3) One sample happy-path flow
4) Notes for next 3 incremental steps

Rules:
- Keep scaffold minimal and extendable.
- Do not over-engineer.
- Add TODO markers for missing business logic.

Output format:
[Structure]
...

[Starter Code]
...

[Next 3 Steps]
1) ...
2) ...
3) ...
```

## Output Format
- Structural proposals
- Starter code
- Next three steps

## Quality Criteria
- Kommer igang snabbt
- Lag startfriktion
- Clear direction for follow-on work

## Variants
- Variant A: API scaffold.
- Variant B: UI/component scaffold.

# Multi Format Doc Export Package

## Purpose
Generate document content that is export-ready for markdown, word, pdf and html.

## Input
- Planinnehall
- Mottagare (dev team, stakeholder, investor)
- Ton/formalitetsniva

## Instructions
Copy the prompt below:

```text
Transform this development plan into multi-format-ready documentation.

Source content:
[insert plan]

Audience:
[insert audience]

Produce:
1) Markdown version (developer-friendly)
2) Executive summary (business-friendly)
3) Print-ready structure notes (for PDF/Word)
4) Web-ready section structure (for HTML)

Rules:
- Keep section IDs/titles consistent across formats
- Add table-friendly blocks for roadmap and risks
- Include appendix-ready prompt library section

Output format:
[Markdown Draft]
...

[Executive Summary]
...

[Print Notes]
...

[HTML Section Map]
...
```

## Output Format
- Multi-format dokumentpaket

## Quality Criteria
- Easy export to multiple formats
- The same factual content in all versions
- Reduces manual rewriting work

## Variants
- Variant A: Internal engineering docs.
- Variant B: Investor/client proposal docs.

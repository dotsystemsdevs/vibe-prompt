# Voice Session: Record, Transcribe, Refine

## Purpose

Standardize voice coding sessions from recording to a finished executable prompt with minimal friction.

## Input

- Session goal
- Recorded transcript
- Project context

## Instructions

Copy the prompt below:

```text
Run this voice coding session workflow:
Record -> Transcribe -> Refine -> Execute

Session goal:
[insert goal]

Transcript:
[insert transcript]

Project context:
[insert context]

Tasks:
1) Produce cleaned transcript.
2) Produce structured coding prompt.
3) Produce execution checklist.
4) Produce post-run reflection notes.

Output format:
[Clean Transcript]
...

[Structured Prompt]
...

[Execution Checklist]
- ...

[Post-Run Notes]
- ...
```

## Output Format

- Clean transcript
- Structured coding prompt
- Checklist and post-run notes

## Quality Criteria

- Consistent session quality
- Less context loss between steps
- Faster path from idea to action

## Variants

- **Variant A:** Solo coding sprint.
- **Variant B:** Pair or mentor voice session.

# Clarify Requirements Before Code

## Purpose
Force clarifying questions and shared understanding before coding begins.

## Input
- Problemstatement
- Affarsmal
- Tekniska ramar

## Instructions
Copy the prompt below:

```text
Before writing code, ask clarifying questions to remove ambiguity.

Context:
[insert context]

Tasks:
1) Ask up to 7 high-impact clarifying questions.
2) Identify assumptions if unanswered.
3) Propose a draft implementation plan based on current info.
4) Mark what could break if assumptions are wrong.

Output format:
[Clarifying Questions]
1) ...
2) ...

[Current Assumptions]
- ...

[Draft Plan]
1) ...
2) ...

[Assumption Risks]
- ...
```

## Output Format
- The question set
- Assumptions
- Draft plan + risks

## Quality Criteria
- Less rework
- Better requirement precision
- Dangerous assumptions are exposed

## Variants
- Variant A: Product feature planning.
- Variant B: Data/ML task scoping.

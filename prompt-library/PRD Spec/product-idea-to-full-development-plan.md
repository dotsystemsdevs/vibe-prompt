# Product Idea To Full Development Plan

## Purpose
Transform a product idea into a complete development plan (product, technology, roadmap, deployment, growth).

## Input
- Product idea
- Mal group
- Doman requirements/regulatory requirements
- Team size and time

## Instructions
Copy the prompt below:

```text
You are an AI Product Manager + Software Architect.
Transform this idea into a complete development plan.

Idea:
[insert idea]

Context:
- Target users:
- Constraints:
- Team size:
- Timeline:

Generate:
1) Product Overview (problem, users, value prop)
2) Technical Solution (stack options + trade-offs)
3) Development Plan (phases, milestones, ownership)
4) Deployment Strategy (environments, CI/CD, monitoring)
5) Growth Strategy (adoption, retention, expansion)

Output format:
[Product Overview]
...

[Technical Solution]
...

[Development Plan]
...

[Deployment Strategy]
...

[Growth Strategy]
...
```

## Output Format
- Complete strategic + technical plan

## Quality Criteria
- Practically wearable
- Clear trade-offs
- Phased delivery with responsibility

## Variants
- Variant A: MVP for 8 weeks.
- Variant B: Enterprise-ready in 6 months.

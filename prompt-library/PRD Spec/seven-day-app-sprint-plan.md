# Seven-Day App Sprint Plan

## Purpose

Turn an app idea from a blank slate to deployment in seven days with clear daily deliverables.

## Input

- App idea
- Preferred stack
- Target customer or user
- Time budget per day

## Instructions

Copy the prompt below:

```text
Create a 7-day app sprint plan for this idea:
[insert idea]

Constraints:
- Time per day: [X]
- Stack: [preferred stack]
- Must include auth, database, API, and UI baseline

For each day provide:
1) Goal
2) Build tasks
3) Done criteria
4) Demo output
5) Risks and fallback

Output format:
[Day 1]
- Goal:
- Tasks:
- Done criteria:
- Demo:
- Risks/fallback:
...
[Day 7]
- Deploy checklist:
- Post-deploy verification:
```

## Output Format

- Day-by-day plan
- Done criteria per day
- Deploy and post-deploy verification checklist

## Quality Criteria

- Sustained pace without chaos
- Practical milestones every day
- Emphasis on shipping, not perfection

## Variants

- **Variant A:** MVP-only (must-haves only).
- **Variant B:** MVP plus polish on day seven.

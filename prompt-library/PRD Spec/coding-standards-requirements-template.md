# Coding Standards and Project Requirements

Locks in your tech stack, patterns, and non-negotiables. Reduces tech debt by keeping the AI consistent across sessions.

## Prompt Template

```
[TECHNICAL REQUIREMENTS]
Language: [e.g. TypeScript]
Framework: [e.g. React 18, Next.js 15]
Patterns: [e.g. functional components only, no class components]
Style guide: [e.g. Airbnb JS, Prettier defaults]
Naming: [e.g. camelCase functions, PascalCase components]
Constraints: [e.g. 80%+ test coverage, WCAG AA accessibility, no external state libs]
File structure: [e.g. feature-based folders under /src/features]

Example:
"All code in TypeScript with React 18. Use functional components only. Follow Airbnb JS style guide. Every function must have type annotations and 80%+ test coverage."
```

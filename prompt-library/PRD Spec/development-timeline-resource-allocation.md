# Development Timeline Resource Allocation

## Purpose
Break down a product into realistic phases with milestones, resource requirements and risk control.

## Input
- Produktscope
- Teamroller
- Deadline
- Risker/beroenden

## Instructions
Copy the prompt below:

```text
Create a phased development timeline with resource allocation.

Project scope:
[insert scope]

Team:
- [roles and count]

Timeline:
[target date or duration]

Tasks:
1) Split into phases (Discovery, Build, Test, Launch, Post-launch)
2) Define milestones per phase
3) Estimate effort per milestone
4) Assign role ownership
5) List top risks + mitigations

Output format:
[Phase Timeline]
- Phase:
  - Duration:
  - Milestones:

[Resource Allocation]
- Role -> responsibility

[Risk Register]
- Risk:
  - Impact:
  - Mitigation:
```

## Output Format
- Fasplan + milstolpar
- Resursallokering
- Riskregister

## Quality Criteria
- Realistisk planering
- Klar ansvarsfordelning
- Tidig riskhantering

## Variants
- Variant A: Solo/founder mode.
- Variant B: Team mode with multiple streams.

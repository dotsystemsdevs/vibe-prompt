# Project Scope and Intent Context

Give the AI the big picture before any code. Keeps the model aligned on WHY you're building, not just what.

## Prompt Template

```
[PROJECT CONTEXT]
Purpose: [What this project/feature is for]
Main goals: [1-3 bullet points]
Users: [Who uses this and how]
Key flows: [The 2-3 main things users do]

Example:
"A React web dashboard to visualize real-time IoT sensor data for factory floor managers. Main flows: live charts, device health, alert management."
```

## Tip

Fill this in once per project and paste it at the top of every new chat session. The AI will stay on-track without you re-explaining context each time.

# Architecture Mermaid Diagram Generator

## Purpose
Create professional Mermaid diagrams for architecture, business flow and timeline directly from text requirements.

## Input
- Systemkomponenter
- Datafloden
- Processer/milstolpar

## Instructions
Copy the prompt below:

```text
Generate Mermaid diagrams from this system description.

System description:
[insert description]

Create:
1) System Architecture diagram
2) Business Flowchart
3) Gantt timeline (phases + milestones)

Rules:
- Keep labels concise
- Use clear directional flows
- Include external integrations and data stores

Output format:
[Architecture Mermaid]
```mermaid
...```

[Business Flow Mermaid]
```mermaid
...```

[Gantt Mermaid]
```mermaid
...```
```

## Output Format
- 3 Mermaid charts ready to paste docs

## Quality Criteria
- Flat load diagram
- Matches text requirements and scope
- Usable in tech/docs/pitch

## Variants
- Variant A: Monolith/system.
- Variant B: Microservices + external APIs.

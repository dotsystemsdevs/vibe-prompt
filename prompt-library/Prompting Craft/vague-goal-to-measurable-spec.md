# Vague Goal To Measurable Spec

## Purpose
Transform vague/unrealistic targets into realistic, actionable delivery criteria.

## Input
- Original moth
- Time frame
- Resource level
- Risk profile

## Instructions
Copy the prompt below:

```text
Translate this goal into a realistic, measurable spec.

Goal:
[insert goal]

Tasks:
1) Identify what is unrealistic or non-measurable.
2) Rewrite into SMART-style criteria.
3) Define acceptance tests.
4) Suggest a phased path (MVP -> improvement).

Output format:
[Unrealistic Parts]
- ...

[Measurable Spec]
- ...

[Acceptance Tests]
- ...

[Phased Path]
- MVP:
- Next:
- Later:
```

## Output Format
- Unrealistic parts highlighted
- Edible requirement spec
- Acceptance tests + phased plan

## Quality Criteria
- Clear expectations
- Reduces dissatisfaction and scope drift
- Easy to verify progress

## Variants
- Variant A: Product template.
- Variant B: Technology template (performance, quality, stability).

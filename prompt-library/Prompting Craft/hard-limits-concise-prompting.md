# Hard Limits Concise Prompting

## Purpose
Get short, precise answers through explicit hard limits instead of vague instructions.

## Input
- Desired length (sentences, bullets, words)
- Prioritization criterion
- Target audience

## Instructions
Copy the prompt below:

```text
Answer with hard limits:
- Max 3 sentences total
- Exactly 5 bullet points
- Max 12 words per bullet

Topic:
[insert topic]

Rules:
- No intro sentence.
- No repetition.
- Highest-impact points first.
```

## Output Format
- Hard-limited svar enligt satta regler

## Quality Criteria
- Predictable length
- Hog signal
- Less prompt friction in iterations

## Variants
- Variant A: Exec summary (3 bullets).
- Variant B: Tech brief (7 bullets + 1 risk line).

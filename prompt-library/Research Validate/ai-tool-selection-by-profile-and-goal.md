# Ai Tool Selection By Profile And Goal

## Purpose
Choose the right AI tooling mix based on experience, budget, project type, and speed requirements.

## Input
- Anvandarprofil
- Projektkategori
- Budget/tidskrav
- Kodvana

## Instructions
Copy the prompt below:

```text
Recommend an AI coding tool stack for my profile and goals.

Inputs:
- Experience:
- Project type:
- Budget:
- Speed vs control preference:

Evaluate categories:
1) Model choice
2) AI editor/IDE
3) Agent platform
4) CLI tools
5) Optional no-code builder

Return:
- Best stack (primary + backup)
- Why this stack fits me
- 7-day onboarding plan
- Cost-saving tips

Output format:
[Recommended Stack]
...

[Why This Fits]
...

[7-Day Onboarding]
Day 1...

[Cost Optimization]
...
```

## Output Format
- Primar + backup stack
- 7-dagars onboarding
- Kostnadsoptimering

## Quality Criteria
- Matchar verklig profil
- Undviker verktygs-overload
- Snabb time-to-output

## Variants
- Variant A: Solo beginner.
- Variant B: Team/pro workflow.

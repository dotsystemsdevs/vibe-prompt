# Monetization Seo Content Operations

## Purpose
Bring the vibe coding product to reality via monetization, SEO and content/distribution.

## Input
- Product type
- Target audience
- Channel strategy
- Revenue goal

## Instructions
Copy the prompt below:

```text
Design a monetization and growth plan for my AI-built product.

Inputs:
- Product:
- Target audience:
- Acquisition channels:
- Revenue target:

Tasks:
1) Propose 2-3 monetization models.
2) Define MVP pricing strategy.
3) Build SEO content plan (topic clusters + publish cadence).
4) Define distribution loop (social/community/email).
5) Add KPI dashboard (traffic, activation, conversion, retention, revenue).

Output format:
[Monetization Options]
...

[Pricing Strategy]
...

[SEO Plan]
...

[Distribution Loop]
...

[KPIs]
...
```

## Output Format
- Monetization options
- Pricing strategy
- SEO and distribution plan
- KPI framework

## Quality Criteria
- From "built" to "used and paid for"
- Data-driven growth logic
- Feasible go-to-market routine

## Variants
- Variant A: Indie solo product.
- Variant B: B2B niche SaaS.

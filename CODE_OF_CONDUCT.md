# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the project and community
- Helping beginners learn

Examples of unacceptable behavior include:

- Harassment, insults, or discrimination
- Trolling or intentionally disruptive behavior
- Publishing private information without consent
- Any abusive or demeaning conduct

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Scope

This code of conduct applies within project spaces (issues, pull requests, discussions, and related community channels).

## Reporting

If you experience or witness unacceptable behavior, open a private report by contacting maintainers through repository communication channels.
